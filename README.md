# Profile Card Design using HTML and CSS

Um projeto simples de **card de perfil** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar um **profile card moderno**, ideal para exibir informações de usuários ou portfólios.

## 🛠️ Tecnologias

- HTML
- CSS

## 🎯 Funcionalidades

- Card de perfil estilizado
- Design simples e moderno
- Fácil de personalizar

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
